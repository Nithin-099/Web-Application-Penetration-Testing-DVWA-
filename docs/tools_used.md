# Tools Used

## Nmap

Purpose:
- Port scanning
- Service discovery
- Version detection

## Burp Suite

Purpose:
- Intercept HTTP requests
- Modify traffic
- Perform manual testing

## Wireshark

Purpose:
- Capture packets
- Analyze HTTP traffic
- Inspect requests and responses

## SQLMap

Purpose:
- Automated SQL Injection testing

## John the Ripper

Purpose:
- Password Cracking

## Nikto

Purpose:
- Web server vulnerability scanning

## Linux Terminal

Purpose:
- Command execution
- Reconnaissance
- Analysis
