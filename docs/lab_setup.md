# Lab Setup

## Environment Overview

A controlled penetration testing laboratory was created to safely perform security testing activities.

## Attacker Machine

### Operating System
Kali Linux

### Purpose
Used for:
- Reconnaissance
- Enumeration
- Exploitation
- Traffic analysis

## Target Machine

### Application
DVWA (Damn Vulnerable Web Application)

### Components
- Apache
- PHP
- MySQL

## Network Configuration

The environment was deployed in an isolated virtual network to prevent unauthorized interaction with external systems.

## Tools Installed

- Nmap
- Burp Suite
- Wireshark
- SQLMap
- Gobuster

## Security Considerations

All testing activities were performed in a controlled and authorized environment.
