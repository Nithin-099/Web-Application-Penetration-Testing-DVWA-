# Testing Workflow

## Step 1 – Environment Setup

- Install DVWA
- Configure database
- Verify application accessibility

## Step 2 – Reconnaissance

- Perform Nmap scans
- Identify open ports
- Enumerate services

## Step 3 – Web Enumeration

- Discover directories
- Analyze application structure
- Identify attack vectors

## Step 4 – Traffic Analysis

- Capture requests using Burp Suite
- Analyze packets using Wireshark

## Step 5 – Vulnerability Testing

Test:
- SQL Injection
- XSS
- CSRF
- File Upload
- Command Injection

## Step 6 – Exploitation

Validate vulnerabilities through controlled exploitation.

## Step 7 – Documentation

Record:
- Evidence
- Screenshots
- Findings
- Recommendations

## Step 8 – Remediation

Provide mitigation strategies and security improvements.
